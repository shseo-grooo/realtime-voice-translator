
## 5. 작업 지시서 (Claude Code 입력용)

> "프로젝트 폴더 내 `whisper-streaming` 라이브러리를 활용하여 실시간 일-한 번역기를 만들어줘.
>
> 1. `whisper_streaming` 패키지의 `WhisperStreaming` 클래스를 사용하여 오디오 스트림을 처리할 것.
> 2. 입력 언어는 일본어(`ja`)로 고정하고, 모델은 `large-v3-turbo`를 사용할 것.
> 3. 인식된 텍스트가 완성될 때마다 로컬 Ollama API를 호출해 한국어로 번역할 것.
> 4. 프론트엔드는 단순한 HTML로 작성하되, 마이크 입력을 실시간으로 서버에 스트리밍해야 해.
> 5. 이전 Faster-Whisper 사용 시 겪었던 환각이나 무한 루프 문제를 방지하기 위해 VAD 설정과 Stability 필터링을 꼼꼼하게 적용해줘.
> 6. 가상환경(venv)을 먼저 만들고 필요한 의존성을 하나씩 설치하면서 진행해줘"

---

### 💡 추가 팁
Claude Code가 코드를 작성할 때, `whisper-streaming`의 의존성 라이브러리(특히 `ctranslate2`) 버전 충돌이 일어날 수 있습니다. Claude에게 **"가상환경(venv)을 먼저 만들고 필요한 의존성을 하나씩 설치하면서 진행해줘"**라고 덧붙이면 훨씬 깔끔하게 작업이 진행될 거예요.
